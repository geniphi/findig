Template Demo

Templates in Findig aren't tied to any particular template engine, although Findig does include a jinja2 based formatter in findig.extras, which was used to render this page. Because the formatter interface is highly customizable however, pretty much any template engine can be used.

The easiest way to add support for additional template engines is to subclass findig.data.TemplateFormatter, which is exactly what findig.extras.JinjaFormatter does (in no more than 50 lines of code!).

The information page has some information about the rendering environment.
